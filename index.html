Professional Profile

I am a systematic, highly hands-on Network Engineer currently operating within the world-class Cisco Customer Experience (CX) Labs environment. I specialize in building, maintaining, and validating complex enterprise-level network architectures and data center compute infrastructures. My daily scope focuses on orchestrating robust physical and virtual test setups, managing complex hardware device inventories, structured cabling typologies, and continuous state configuration management.

Driven by rigorous network verification and empirical diagnostics, I excel in executing comprehensive validation plans and performance testing under high-load conditions using leading evaluation systems like Ixia and Spirent. I possess strong capabilities in cross-layer troubleshooting and replication of critical network infrastructure bugs in isolated lab environments to perform highly accurate Root Cause Analysis (RCA). My deep routing stack expertise includes deploying Layer 1-4 topologies, precise VLAN segmentation, sub-networking schemes, and dynamic routing architectures running OSPF, EIGRP, and RIP.

Furthermore, my comprehensive skillset spans infrastructure security hardenings and enterprise storage virtualization. I am experienced in provisioning and configuring Cisco Next-Generation security appliances (Cisco ASA and Firepower/FTD hardware) through FMC and FDM platforms—rigorously validating ACL policies, complex NAT setups, and secure IPsec VPN tunnels. My system administration competencies extend to managing scalable virtual environments on VMware ESXi hypervisors, alongside bare-metal hardware orchestration and server provisioning within Cisco UCS platforms using CIMC.
