Professional Profile

I am a detail-oriented Network Engineer with hands-on experience in building, maintaining, and validating complex enterprise and lab network environments. My expertise spans the implementation of robust Layer 1-4 architectures, utilizing core routing protocols (OSPF, EIGRP, RIP), VLAN segmentation, and precise IP addressing. I possess a strong background in network security and virtualization, deploying and configuring Cisco NGFW solutions (ASA, Firepower/FPR via FMC/FDM) alongside managing virtualized infrastructures on VMware ESXi and hardware provisioning within Cisco UCS environments.


Driven by technical accuracy, I specialize in advanced troubleshooting and performance validation, executing comprehensive test plans using industry-standard traffic generators like Ixia and Spirent, and performing deep packet-level analysis via Wireshark for Root Cause Analysis (RCA). Dedicated to operational continuity, I excel at producing structured technical documentation, ensuring complex network topologies and test outcomes are clearly articulated for seamless knowledge sharing.
